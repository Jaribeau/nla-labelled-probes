# Is Evaluation Awareness Just Format Sensitivity? (Devbunova 2026)

- **Paper:** arXiv 2603.19426
- **Code:** https://github.com/viliana-dev/eval-awareness-format
- **Dataset:** https://huggingface.co/datasets/viliana-dev/eval-awareness-2x2
- **Venue:** ICLR 2026 workshop

## Why it matters here

Direct conceptual parallel. Devbunova showed linear probes for eval-awareness primarily detect **benchmark formatting** rather than genuine awareness. Our project asks the analogous question for the LLM realism judge: does it have similar feature sensitivities (stakes, arc, coherence) rather than tracking "evalness" per se?

The public **2×2 dataset** is a candidate pilot for direct probe-vs-judge comparison: run the same transcripts through Petri's realism judge and compare to Devbunova's probe classifications.
