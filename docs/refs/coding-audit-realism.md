# Measuring and improving coding audit realism

- **Link:** https://alignment.anthropic.com/2026/coding-audit-realism/
- **Source:** Anthropic Alignment, March 2026

## Why it matters here

Defines the **realism win rate** metric: how often an LLM judge falsely identifies an eval transcript as the "real" one when paired with a deployment transcript. Giving the auditor real system prompts/tools raised win rate from **4.6% → 32.8%**. The appendix flags a **stakes heuristic** (judge sensitivity to scenario severity) — this is the basis for our Stakes validation experiment.
